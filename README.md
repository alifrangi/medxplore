## MedXplore

A medical excellence and innovation platform connecting students, departments, and administrators.
- **Student Passport System** - Track achievements and engagement
- **Department Dashboards** - Manage teams, files, and communications
- **Event Management** - Organize and discover medical events
- **News & Updates** - Stay informed with the latest workshops and events
